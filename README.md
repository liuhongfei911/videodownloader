# 视频下载器

一个功能强大的视频下载工具，支持多种视频平台的视频解析和下载，并提供友好的用户界面和丰富的下载选项。

## 功能特点

- 🔐 密码保护：需要输入正确密码才能使用
- 🎬 多平台支持：支持解析多个视频平台的链接
- 📥 批量下载：支持同时下载视频封面和视频文件
- 📊 进度显示：实时显示下载进度、速度和剩余时间
- 🔄 重试机制：下载失败时自动重试，提高成功率
- 🎨 现代UI：简洁美观的用户界面，流畅的交互体验
- 📱 响应式设计：适配不同屏幕尺寸

## 技术特点

- 强化的下载处理：特别针对图片文件优化，确保直接下载而非在浏览器中打开
- URL预处理：智能处理相对URL和特殊参数，提高下载成功率
- 文件名清理：自动处理特殊字符，生成合法文件名
- 通知系统：友好的通知提示，支持手动关闭
- 并发控制：智能调整并发下载数量，避免服务器压力过大

## 使用方法

1. 打开应用后，输入密码（默认为`123456`）
2. 在输入框中粘贴视频链接
3. 点击"解析"按钮获取视频信息
4. 预览视频内容和封面
5. 选择下载选项：
   - 下载封面：仅下载视频封面图片
   - 下载视频：仅下载视频文件
   - 下载全部：同时下载封面和视频

## 开发说明

### 本地运行

1. 克隆或下载项目代码
2. 使用本地服务器运行项目（如Python的http.server或Node.js的http-server）
3. 在浏览器中访问项目地址

### 技术栈

- 纯HTML/CSS/JavaScript实现，无需外部依赖
- 使用现代Web API实现下载功能
- 响应式设计，适配多种设备

### 自定义配置

- 修改密码：在`app.js`中修改`CORRECT_PASSWORD`常量
- 调整API：在`app.js`中修改`API_URL`和`API_KEY`常量
- 自定义样式：在`styles.css`中修改CSS规则
